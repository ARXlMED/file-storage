# file-storage
## Что отражено в отчёте? (получился очень большой поэтому краткое описание)
Присутсвует демонстрация абсолютно всех команд, для различных видов файлов, также отражены все пакеты в wireshark в общем и конкретно http-запросы и ответы для каждого из методов.
## What is this application? / Что из себя представляет данное приложение?
Данное приложение это файловое хранилище работающее на lookback адресе 127.0.0.2 и осуществляющая приём команд и пересылку файлов через протокол http.
## Start Working / Начало работы
1. Открыть папку App и запустить приложение
2. Открыть консоль и вписывать команды curl, также можно отправлять запросы по браузеру. Примеры команд в curl с пояснениями приведены в текстовом документе "Commands in curl.txt"
